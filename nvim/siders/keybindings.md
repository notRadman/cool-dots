Neovim Keymap Reference (Clean Text)
Leader key → ,

==== General ====

== Window Navigation
Ctrl + h → move left

Ctrl + l → move right

Ctrl + j → move down

Ctrl + k → move up

== Window Resizing
Space + h → decrease width (left)

Space + l → increase width (right)

Space + j → decrease height (down)

Space + k → increase height (up)


==== Nvim-Tree (File Explorer) ====

== Global
,e → toggle file tree

,o → focus file tree

,i → return to previous window

,ff → reveal current file in tree

,re → toggle tree side (left/right)

== Inside Tree
C → change root to selected folder

U → go up to parent

? → help menu

Default tree keys → Enter (open), a (new), d (delete), r (rename), etc.


==== Telescope ====

,ff → find files

,fg → live grep

,fb → list/search buffers


==== ToggleTerm ====

,t → toggle terminal

Esc → normal mode from terminal


==== Extra ====
,w → close all side tools (tree + terminal)


==== Default Nvim-Tree Keys (Quick List) ====

Enter / o → open

a → create

d → delete

r → rename

x → cut

c → copy

p → paste

y → copy filename

Y → copy relative path

gy → copy full path

- → parent directory

s → horizontal split

v → vertical split

t → new tab

? → help
